# Code of Conduct

## Our Standards

We are committed to providing a welcoming and inclusive environment for everyone.

Expected behavior:
- Be respectful and constructive in discussions
- Accept feedback gracefully
- Focus on what is best for the community and the project

Unacceptable behavior:
- Personal attacks or derogatory comments
- Harassment in any form
- Publishing others' private information without permission

## Scope

This code of conduct applies to all project spaces including issues, pull requests, and discussions.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not align with this code of conduct. Instances of unacceptable behavior may be reported to the project maintainers.

This code of conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/).
